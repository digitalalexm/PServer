-calcftrdist
template: commu?clnt=<name|pass>&com=calcftrdist&smetric=<metric>&whr=<whr>>
descript: Calculates the distances between features. The metric that will be used is passed as a parameter
	  and must be a valid pbean name that is loaded by the PServer. The whr parameter is optional and defines the pattern
	  of feature names that you want to calculate the distances.
example : commu?clnt=tname|pass&com=calcftrdist&smetric=cos
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK  : no response body exists.
