-addftr template 
-rmftrgrp
template: commu?clnt=<name|pass>&com=rmftrgrp&grp=<name>
descript: Removes a feature group from the database. The parameter grp is 
 	  the name of the feature group that we need to remove
example : commu?clnt=test1|test1&com=rmftrgrp&grp=grp1
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK : no response body exists.
