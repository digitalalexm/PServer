-rmfusrcom
template: commu?clnt=<name|pass>&com=rmfusrcom&ucom=<name>
descript: Removes a user community from the database. The parameter ucom is the name of the user community that 
 	  we need to remove. 
example : commu?clnt=test1|test1&com=rmfusrcom&ucom=mycommunity
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK : no response body exists.
