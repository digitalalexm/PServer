-getusrcomu
template: commu?clnt=<name|pass>&com=getusrcomu&ucom=<name>
descript: Returns the users of a specific user community. 
example : commu?clnt=test1|test1&com=rmfusrcom&ucom=mycommunity
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK  : in this case the response body is as follows
	<?xml version="1.0"?>
	<?xml-stylesheet type="text/xsl" href="/resp_xsl/community_users.xsl"?>
	<result>
	<row><user>feature</user></row>
	...
	</result>
