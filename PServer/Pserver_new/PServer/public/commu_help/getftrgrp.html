-getftrgrp
template: commu?clnt=<name|pass>&com=getftrgrp&ftrgrp=<name1>
descript: Returns the features of a feature group. The parameter ftrgrp contains the name of the feature group. 
returns : 200 OK, 401 (fail, request error), 501 (fail, server error)
200 OK  : in this case the response body is as follows
	<?xml version="1.0"?>
	<?xml-stylesheet type="text/xsl" href="/resp_xsl/group_features.xsl"?>
	<result>
	<row><feature>feature</feature></row>
	...
	</result>
